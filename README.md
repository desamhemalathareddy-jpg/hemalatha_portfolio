# 🌐 Personal Portfolio Website

Welcome to my personal portfolio website! This project showcases my skills, education, projects, and achievements as a Computer Science Engineering (Artificial Intelligence & Machine Learning) student. It is designed with a modern, responsive, and interactive user interface to provide visitors with an engaging experience.

## 🚀 About the Project

This portfolio represents my journey as an aspiring Software Engineer. It highlights my technical skills, academic background, project experience, and contact information in a clean and professional layout. The website is fully responsive and includes smooth animations to enhance the user experience.

## ✨ Features

- Responsive design for all devices
- Modern and professional UI
- Animated typing effect
- Smooth scrolling navigation
- Skills section with animated progress bars
- Education timeline
- Project showcase
- Contact section
- Interactive hover animations
- Clean and organized code structure
